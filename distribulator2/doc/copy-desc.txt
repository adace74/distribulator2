Copy Command Syntax & Examples
-----------------------------

Syntax: copy [[source [server]]:source_path [[destination [server]]:dest_path

Description: Copies files to/from various servers within our
             working environment.

Examples:
    copy /tmp/blah.txt /tmp/
    - Copies local file /tmp/blah.txt to directory /tmp/ 
      on the current server group.
    copy /tmp/blah.txt www:/tmp/
    - Copies local file /tmp/blah.txt to directory /tmp/
      on the 'www' server group.
