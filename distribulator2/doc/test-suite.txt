The idea here is to try and verify that all syntax checking, and logical execution
is in fact acting properly.

######################################################################

Command(s): copy

Test Case:       copy
Expected Result: Error.

Test Case:       copy /tmp/test.out /var/tmp/
Expected Result: Copies local /tmp/test.out file out to all current server group
                 servers, remote directory /var/tmp/.

Test Case:       copy /tmp/test.out www
Expected Result: Error.

Test Case:       copy /tmp/test.out www:/var/tmp/
Expected Result: Copies local /tmp/test.out file out to all 'www' server group servers,
		 remote directory /var/tmp/.

Test Case:       copy /tmp/test.out www01st:/var/tmp/
Expected Result: Copies local /tmp/test.out file out to the 'www01st' server,
		 remote directory /var/tmp/.

######################################################################

Command(s): help

Test Case:       help
Expected Result: Main Help Page.

Test Case:       help <value>, where value is each command supported.
Expected Result: Detailed help page.

Test Case:       help jfjfjf
Expected Result: Main Help Page.

######################################################################

Command(s): login

Test Case:       login
Expected Result: Error.

Test Case:       login www01st
Expected Result: SSH Connection to www01st.

Test Case:       login jfjfjf
Expected Result: Error.

######################################################################

Command(s): run

Test Case:       run
Expected Result: Error

Test Case:       run "uptime"
Expected Result: Runs the "uptime" command on current server group.

Test Case:       run "uptime" www
Expected Result: Runs the "uptime" command on 'www' server group.

Test Case:       run "uptime" www01st
Expected Result: Runs the "uptime" command on the 'www01st' server host.

######################################################################

Command(s): server-group and server-list

Test Case:       server-group
Expected Result: Error.

Test Case:       server-group www
Expected Result: Changes current group to 'www'.

Test Case:       server-list
Expected Result: Output conforms to the interaction of both commands.
