The idea here is to try and verify that all syntax checking,
and logical execution is in fact acting properly.

######################################################################

Command(s): copy

Test Case:       copy
Expected Result: Error.

Test Case:       copy /tmp/bogus /tmp/
Expected Result: Error.

Test Case:       copy /tmp/test.out /tmp/
Expected Result: Copies local /tmp/test.out file out to current server group, 
                 remote directory /tmp/.

Test Case:       copy /tmp/test.out www
Expected Result: Error.

Test Case:       copy /tmp/test.out www:/tmp/
Expected Result: Copies local /tmp/test.out file out to all 'www' servers,
		 remote directory /tmp/.

Test Case:       copy /tmp/test.out www01st:/tmp/
Expected Result: Error.

######################################################################

Command(s): help

Test Case:       help
Expected Result: Main Help Page.

Test Case:       help <value>, where value is each command supported.
Expected Result: Detailed help page.

Test Case:       help jfjfjf
Expected Result: Error.

######################################################################

Command(s): login

Test Case:       login
Expected Result: Error.

Test Case:       login www01st
Expected Result: SSH Connection to www01st.

Test Case:       login jfjfjf
Expected Result: Error.

######################################################################

Command(s): run

Test Case:       run
Expected Result: Error.

Test Case:       run "uptime"
Expected Result: Runs the "uptime" command on current server group.

Test Case:       run "uptime" on www
Expected Result: Runs the "uptime" command on 'www' server group.

Test Case:       run "uptime" on www01st
Expected Result: Error.

######################################################################

Command(s): server-group and server-list

Test Case:       server-group
Expected Result: Listing of available server groups.

Test Case:       server-group www
Expected Result: Sets current group to 'www'.

Test Case:       server-list
Expected Result: Listing of servers in current server group.

Test Case:       server-group www
Expected Result: Listing of servers in 'www' server group.
